### B. Strategic decisions LOCKED IN

**1. IP defense reality** — heuristics alone are not patent-eligible (Alice/Mayo); copyright covers literal copy only; trade-secret is incompatible with BSL-1.1 publishing. **The moat is the SYSTEM around the heuristics**: audit-chain spec + policy.json rule library + community learnings dataset + integration breadth. Documented in [docs/ROADMAP.md](../ROADMAP.md).

**2. Shared learnings tiering** — community library is **PRO+ feature, NOT free**. Locked in [docs/architecture/shared-learnings-tiering.md](../architecture/shared-learnings-tiering.md):
- Community (free): local-only learnings (today's behavior)
- Pro: read access to community library (~10K+ vetted rules at maturity)
- Team: read+write team-private library via `policy.json` `extends:`
- Enterprise: org-wide + SSO + (eventually) SOC 2

This is the **data network effect** — every miss saved by every user makes every future user smarter. Single biggest unbuilt moat. Pricing rebalance proposed: CHF 2 → 5-9 / month for Pro.

**3. SOC 2 + SSO are deal-triggered, not date-triggered** — user explicitly said *"I will not pay for the trust certificate"*. Phase 4 spend ($30K-$50K) is parked until a written LOI demands it. Foundations (CodeQL, Dependabot, audit log, policy.json) accrue evidence for free in the meantime.

**4. The LLM-judge layer is Phase 2** — not Phase 1. `policy.json` extends with a `judges:` array; users author framework-specific judges (Flutter spatial-fit, React a11y, etc.). OpsContext ships the engine, users ship the prompts. Locked in [docs/ROADMAP.md](../ROADMAP.md).

**5. Spatial-fit / a11y / framework auditing is NOT built into OpsContext** — wrong abstraction. Ship the policy-driven LLM-judge instead. Explicitly in the anti-roadmap.

**6. Browser capture wedge is REAL** — workflow found 8 Chrome extensions that export AI conversations to files (top: Couchraver/claude-chatgpt-gemini-downloader 14⭐, MIT). NONE stream into a tamper-evident audit log; NONE deliver terminal alerts; NONE enforce policy. The two MIT-licensed repos are usable as a selector library starting point with attribution.
