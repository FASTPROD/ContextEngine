### Commit `c68968e` ÔÇö `feat(audit): tamper-evident hash-chained audit log`
The compliance bedrock ÔÇö P0 #3 part 1.

- **New file `src/audit.ts`** ÔÇö `AuditLogger` with hash-chained JSONL
  at `~/.contextengine/audit.log`. Each record:
  `{ts, event, actor, payload, prev_hash, hash}`. Genesis = 64 zeros.
  SHA-256 over canonical serialization `{prev_hash, ts, event, actor, payload}`
  in that fixed key order. Any historical mutation breaks verification
  at the mutated index.
- **`safeAppend()`** wrapper isolates audit failures from production
  hot paths ÔÇö a failed append logs to stderr only, never throws upward
  (production hot paths in learnings/sessions/activation use this; tests
  and CLI use `appendAudit()` directly).
- **Env-var path injection** ÔÇö `CONTEXTENGINE_HOME` overrides
  `homedir()/.contextengine`. Tests run in `mkdtempSync()` dirs and
  cannot pollute the real `~/.contextengine`.
- **Wired into** the LOCKED files but only at the entry/exit boundaries
  (did not touch the locked algorithms): `learnings.ts` saveLearning
  (create + update branches), deleteLearning, importLearningsFromFile
  (per-entry + aggregate import event); `sessions.ts` saveSession,
  deleteSession; `activation.ts` activate, deactivate.
- **Privacy by construction** ÔÇö records carry metadata only (ID,
  category, project, lengths). Never the rule text, session value
  content, or license signature.
- **`audit_verify` MCP tool** ÔÇö agents can self-check chain integrity.
  Brings tool count to **20** (was 19 after `delete_session` registration).
- **CLI** ÔÇö `audit-export [--since DATE] [--until DATE] [--format jsonl|csv]`
  and `audit-verify` (exit code 2 on broken chain ÔÇö CI/cron monitoring).
- **`tests/audit.test.ts`** ÔÇö 19 new tests covering genesis, multi-record
  chain linking, payload tampering, prev_hash splicing (record removed
  from middle), appended-forgery (bogus prev_hash), corrupt JSON graceful
  failure, range filtering, RFC 4180 CSV escaping verified by quote-count
  invariant. **100 / 100 tests pass.** 10 files changed: +640 / Ôłĺ14.
- **Smoke-tested end-to-end**: sandboxed `CONTEXTENGINE_HOME=$tmpdir`,
  save a learning, `audit-verify` Ôćĺ Ôťů 1 record intact, JSONL + CSV
